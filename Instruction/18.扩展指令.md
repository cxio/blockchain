# 扩展指令

通用的扩展指令区，用于支持更多的可能性。

该部分指令不属于基础指令集范畴，但依然是灵活性的必要补充。


## 定义

指令码区间：`[251-253]`，3 个。

```go
码值    指令字          说明
-----------------------------------------------------------
251     EXT_MO(2){}     扩展模块包
        // 附参：目标索引。
        // 专门的扩展模块空间，64k指令位。

252     （未用）

253     EXT_PRIV(2){}   私有扩展包
        // 附参：目标索引。
        // 私有：
        // 为专门的私有需求提供，自由定制。
        // 公共校验的逻辑不会抵达此处，
        // 如果在END/INPUT之前出现，脚本会被视为错误而失败。
```

> **注：**
> 最高码值 `254/255` 不占用，为系统保留，不在指令集范畴。
