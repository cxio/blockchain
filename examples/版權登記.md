## 版權登記

### 用例

版權聲明類似於存在性證明，但版權登記通常是監管繫統的要求。與存在性證明類似，版權聲明同樣需要「交易過期」特性的支持。


### 操作

類似於存在性證明，版權聲明分兩步執行。

1. 先發布一筆版權聲明交易（管理者交易），提供版權物說明和附件ID的合並哈希摘要。
2. 版權聲明交易進入區塊並等到交易過期時段後，發布一筆憑信交易，包含版權物說明和可能有的附件，並包含跳轉到監管腳本的邏輯。
3. 轉移該版權憑信（花費）。如果版權實際上無需轉移，可置位憑信描述配置裏的銷毀標記。


### 腳本

#### 版權聲明（管理者交易）

```go
IN{Receiver}                    // 接收者（僅登記需要）
IN{Description}                 // 取版權物說明入棧
IN{Attachment}                  // 取附件ID入棧

@TOPS[2]                        // 引用棧頂2項為實參
ADD                             // 內容合並
FN_HASH256                      // 計算哈希摘要
DATA{<hashResult>} EQUAL PASS   // 與哈希結果比較，相等則通過

IN{Script}                      // 取當前腳本片段入棧（監管驗證）

@POPS[4]                        // 取出棧頂4項（接收者、說明、附件ID、腳本片段）
GOTO[<Height>, <TxID>, <n>]     // 跳轉到監管者的登記腳本
```


#### 監管登記（管理者交易）

需要驗證版權聲明腳本格式的合法性，否則無法區分登記信息是否為偽造者搶註。

```go
ENV{FromGOTO} PASS                  // 應當由GOTO跳轉過來

@POP                                // 取出源腳本（版權聲明）
MODEL<PRINT>{                       // 模型驗證
    IN{Receiver}                    // 嚴格匹配
    IN{Description} IN{Attachment}  // 嚴格匹配
    @TOPS[2] ADD FN_HASH256         // 嚴格匹配
    DATA[32]{?} EQUAL PASS          // 哈希結果任意，但等長驗證
    IN{Script}                      // 嚴格匹配
    @POPS[4]                        // 嚴格匹配
    GOTO[<Height>, ?, <n>]          // 當前腳本信息（TxID 未知）
}                                   // 模型驗證通過

@POPS[3]                            // 取登記信息（接收者、版權物說明、附件ID）
OUTPUT BUFDUMP                      // 導出並觸發監管登記進程
```


#### 版權物憑信

```go
// 版權聲明腳本定位
{<Height>}                  // 理想塊高度
DATA{<TxID>}                // 交易ID
{<OutIndex>}                // 輸出序位
JUMP                        // 跳轉到版權聲明交易（自動返迴）

@TEXT{Its done.} PRINT      // 友好打印
@ENV{TxID} OUTPUT BUFDUMP   // 取交易ID導出，或觸發外部應用（可選）
```

> **說明：**<br>
> 在該憑信輸出中寫入版權的內容和版權物附件的ID，跳轉到版權聲明處驗證。<br>
> 需要花費掉（轉移）該憑信，才會執行公共域的監管者登記腳本。<br>


### 腳本II（嚴格匹配）

在版權聲明腳本的模型驗證中，如果需要嚴格匹配 `GOTO` 指令中的監管登記交易，可以將模型驗證放在外部，卽另一筆管理者交易中。


#### 版權聲明II

```go
// （同前版權聲明）
// IN{Receiver}
// IN{Description}
// IN{Attachment}
// @TOPS[2]
// ADD
// FN_HASH256
// DATA{<hashResult>} EQUAL PASS
// IN{Script}

@POPS[7]                        // 取出棧頂7項（含模型驗證腳本定位）
GOTO[<Height>, <TxID>, <n>]     // 跳轉到監管登記II腳本
```


#### 監管登記II（交易ID確定）

```go
ENV{FromGOTO} PASS      // 應當由GOTO跳轉過來

@GETS[0,3]              // 提取模型驗證腳本定位信息
JUMP                    // 跳轉到模型驗證腳本

@POPS[3]                // 取登記信息（接收者、版權物說明、附件ID）
OUTPUT BUFDUMP          // 導出並觸發監管登記進程
```


#### 模型驗證

```go
@POP                                // 取出源腳本（版權聲明）
MODEL<PRINT>{                       // 模型驗證
    IN{Receiver}                    // 嚴格匹配
    IN{Description} IN{Attachment}  // 嚴格匹配
    @TOPS[2] ADD FN_HASH256         // 嚴格匹配
    DATA[32]{?} EQUAL PASS          // 哈希結果任意，但等長驗證
    IN{Script}                      // 嚴格匹配
    @POPS[7]                        // 嚴格匹配
    GOTO[<Height>, <TxID>, <n>]     // 監管登記II腳本信息
}
```


#### 版權物憑信II

```go
// 模型驗證腳本定位
{<Height>}                  // 理想塊高度
DATA{<TxID>}                // 交易ID
{<OutIndex>}                // 輸出序位

// （同前版權物憑信）
// {<Height>}
// DATA{<TxID>}
// {<OutIndex>}
// JUMP
// @TEXT{Its done.} PRINT
// @ENV{TxID} OUTPUT BUFDUMP
```


### 腳本III（嚴格匹配）

其實，最好的做法是從環境變量 `ENV{FromSource}` 中獲取跳轉前的源碼檢測。這是一個專門用於解決上面「模式無法獲取自身交易ID」問題的變量，它的定義是：跳轉來源腳本中，跳轉指令或行為（JUMP會自動返迴）之前的指令序列，不含跳轉指令本身。


#### 版權聲明

```go
// （同前版權聲明）
// IN{Receiver}
// IN{Description}
// IN{Attachment}
// @TOPS[2]
// ADD
// FN_HASH256
// DATA{<hashResult>} EQUAL PASS

@POPS[3]                        // 取出棧頂3項（接收者、說明、附件ID）
GOTO[<Height>, <TxID>, <n>]     // 跳轉到監管者的登記腳本
```


#### 監管登記（管理者交易）

```go
ENV{FromGOTO} PASS                  // （同前）
@ENV{FromSource}                    // 取出跳轉前段腳本

MODEL<PRINT>{                       // 模型驗證
    IN{Receiver}                    // 嚴格匹配
    IN{Description} IN{Attachment}  // 嚴格匹配
    @TOPS[2] ADD FN_HASH256         // 嚴格匹配
    DATA[32]{?} EQUAL PASS          // 哈希結果任意，但等長驗證
    @POPS[3]                        // 嚴格匹配
    // GOTO[...]                    // 跳轉點必然存在，無需匹配。
}

@POPS[3]                            // （同前）
OUTPUT BUFDUMP                      // （同前）
```


#### 版權物憑信

```go
// （同前）
// 版權聲明腳本定位
// {<Height>}
// DATA{<TxID>}
// {<OutIndex>}
// JUMP
// @TEXT{Its done.} PRINT
// @ENV{TxID} OUTPUT BUFDUMP
```
